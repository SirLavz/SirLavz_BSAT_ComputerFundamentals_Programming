1. Course Description

This course introduces Air Transportation students to the fundamental concepts of computer hardware, software, operating systems, and networking. It then transitions into the principles of programming, emphasizing algorithmic thinking, problem-solving, and structured programming. A significant portion of the course is dedicated to object-oriented programming using Java, focusing on its application in developing logical solutions and simple tools relevant to the aviation industry. Students will gain practical skills in writing, debugging, and executing Java programs, preparing them to understand and potentially contribute to the technological aspects of modern air transportation systems.

2. Course Objectives / Learning Outcomes

Upon successful completion of this course, students will be able to:

  • CO1: Understand the basic components and functions of computer hardware, software, and operating systems.
  • CO2: Explain fundamental networking concepts and their relevance in modern communication systems, particularly in aviation.
  • CO3: Apply logical thinking and problem-solving techniques to design algorithms using flowcharts and pseudocode.
  • CO4: Write, compile, and execute basic Java programs utilizing fundamental syntax, data types, and control structures.
  • CO5: Implement Object-Oriented Programming (OOP) principles (encapsulation, inheritance, polymorphism, abstraction) in Java.
  • CO6: Develop simple Java applications that demonstrate file handling, basic data structures, and user interface elements.
  • CO7: Relate programming concepts and computational thinking to real-world scenarios and potential applications within the air transportation sector.

3. Course Content / Topics

This course will cover the following topics, structured over a [15+3] week period:

Module 1: Introduction to Computers and Information Technology (Weeks 1-3)

  • **Topic 1.1:** Introduction to Computers
    • Definition, History, Generations of Computers
    • Types of Computers (Supercomputer, Mainframe, Minicomputer, Microcomputer)
    • Components: Hardware vs. Software
  • **Topic 1.2:** Computer Hardware
    • Input Devices (Keyboard, Mouse, Scanners, Microphones)
    • Output Devices (Monitors, Printers, Speakers)
    • Central Processing Unit (CPU): Components and Functions
    • Memory (RAM, ROM, Cache)
    • Storage Devices (HDD, SSD, USB drives, Cloud Storage)
  • **Topic 1.3:** Computer Software
    • System Software (Operating Systems, Utility Programs, Device Drivers)
    • Application Software (Word Processors, Spreadsheets, Databases, Aviation-specific software examples)
  • **Topic 1.4:** Operating Systems
    • Functions of an OS
    • Types of OS (Windows, macOS, Linux, Mobile OS)
    • Basic OS commands and file management
  • **Topic 1.5:** Computer Networks and Internet
    • Network Topologies (LAN, WAN)
    • Network Devices (Routers, Switches, Hubs)
    • Protocols (TCP/IP, HTTP, FTP)
    • Introduction to the Internet and its services
    • Relevance in Air Traffic Management (ATM) and airline operations

Module 2: Fundamentals of Programming and Algorithmic Thinking (Weeks 4-6)

  • **Topic 2.1:** Introduction to Programming
    • What is Programming? Why Program?
    • Programming Languages (High-level vs. Low-level)
    • Compilers and Interpreters
  • **Topic 2.2:** Problem Solving and Algorithms
    • Steps in Problem Solving
    • Definition of Algorithm Characteristics
    • Developing Algorithms for simple problems
  • **Topic 2.3:** Pseudocode
    • Rules and conventions for writing pseudocode
    • Translating simple problems into pseudocode
  • **Topic 2.4:** Flowcharting
    • Standard Flowchart Symbols
    • Drawing Flowcharts for various logical constructs (sequence, selection, iteration)
    • Tracing Flowcharts

Module 3: Introduction to Java Programming (Weeks 7-9)

  • **Topic 3.1:** Getting Started with Java
    • History and Features of Java
    • Java Development Kit (JDK), Java Runtime Environment (JRE), Java Virtual Machine (JVM)
    • Setting up a Java Development Environment (IDE like IntelliJ IDEA, Eclipse, or VS Code)
    • Writing and Compiling your First Java Program (`Hello World`)
  • **Topic 3.2:** Java Basics: Syntax and Structure
    • Comments, Keywords, Identifiers
    • Basic Program Structure (`public static void main(String[] args)`)
  • **Topic 3.3:** Data Types and Variables
    • Primitive Data Types (int, double, char, boolean, etc.)
    • Declaring and Initializing Variables
    • Type Casting
  • **Topic 3.4:** Operators
    • Arithmetic, Relational, Logical, Assignment, Increment/Decrement Operators
    • Operator Precedence
  • **Topic 3.5:** Input and Output
    • Using `System.out.println()` and `System.out.print()`
    • Reading User Input with `Scanner` Class
  • **Topic 3.6:** Control Structures
    • **Selection Statements:** `if`, `if-else`, `if-else if-else`, `switch`
    • **Looping Statements:** `while`, `do-while`, `for` loops
    • `break` and `continue` statements

Module 4: Object-Oriented Programming (OOP) in Java (Weeks 10-13)

  • **Topic 4.1:** Introduction to OOP Concepts
    • Objects and Classes
    • Advantages of OOP
  • **Topic 4.2:** Classes and Objects
    • Defining Classes (Attributes/Fields, Methods)
    • Creating Objects
    • Constructors (Default, Parameterized)
    • `this` Keyword
  • **Topic 4.3:** Encapsulation
    • Access Modifiers (`public`, `private`, `protected`, default)
    • Getters and Setters
  • **Topic 4.4:** Inheritance
    • `extends` Keyword
    • `super` Keyword
    • Method Overriding
  • **Topic 4.5:** Polymorphism
    • Method Overloading
    • Runtime Polymorphism (Dynamic Method Dispatch)
  • **Topic 4.6:** Abstraction
    • Abstract Classes and Abstract Methods
    • Interfaces
  • **Topic 4.7:** Packages and JAR Files
    • Organizing classes into packages
    • Creating and using JAR files

Module 5: Advanced Java Concepts and Applications (Weeks 14-16)

  • **Topic 5.1:** Arrays
    • One-dimensional Arrays
    • Multi-dimensional Arrays
    • Array of Objects
  • **Topic 5.2:** String Handling
    • `String` Class and its methods
    • `StringBuilder` and `StringBuffer`
  • **Topic 5.3:** Exception Handling
    • `try-catch-finally` block
    • `throw` and `throws` keywords
    • Custom Exceptions
  • **Topic 5.4:** File Input/Output (I/O)
    • `File` Class
    • Reading from and Writing to Text Files (`FileReader`, `FileWriter`, `BufferedReader`, `BufferedWriter`)
    • Introduction to Binary File I/O (Optional/Brief)
    • *Practical Application:* Reading/writing flight data logs, passenger manifests (simplified).
  • **Topic 5.5:** Introduction to Graphical User Interface (GUI) with Swing/JavaFX (Basic)
    • Creating simple windows, buttons, text fields
    • Event Handling (Action Listeners)
    • *Practical Application:* Simple flight status display, calculator for aviation metrics.
  • **Topic 5.6:** Introduction to Data Structures (Brief Overview)
    • Lists (ArrayList)
    • Maps (HashMap)
  • **Topic 5.7:** Programming Applications in Air Transportation
    • Discussion of how programming is used in:
      • Air Traffic Control (ATC) systems
      • Flight Planning and Optimization
      • Aircraft Maintenance and Diagnostics
      • Airline Reservation Systems
      • Crew Scheduling
      • Simulators
    • Mini-Project: Students will propose and develop a small Java application related to air transportation (e.g., a simple flight time calculator, a basic passenger manifest manager, a unit converter for aviation metrics, a simple aircraft weight and balance tool).

4. Teaching Methodologies

The course will employ a blended approach to facilitate effective learning:

  • Lectures: Explanations of theoretical concepts, principles, and syntax.
  • Hands-on Laboratory Sessions: Practical exercises, coding drills, and guided programming tasks.
  • Problem-Based Learning: Students will be presented with real-world (or simulated) aviation-related problems to solve using programming.
  • Demonstrations: Live coding and demonstration of software tools and programming techniques.
  • Collaborative Learning: Group activities and peer programming for complex tasks.
  • Project-Based Learning: A culminating mini-project requiring application of learned concepts.

5. Assessment / Grading System

Student performance will be evaluated based on the following components:

  • Quizzes (20%): Short assessments to check understanding of theoretical concepts and syntax.
  • Laboratory Exercises / Assignments (30%): Practical coding tasks, problem-solving assignments, and regular programming exercises.
  • Midterm Examination (20%): Covers Modules 1-3.
  • Final Examination (20%): Comprehensive, with emphasis on Modules 4-5.
  • Mini-Project (10%): Design, implementation, and presentation of a small Java application relevant to air transportation.
