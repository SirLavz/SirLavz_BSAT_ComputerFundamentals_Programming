COMP 213 - COMPUTER FUNDAMENTALS AND PROGRAMING

Java Aviation Information System

Problem Description

Your task is to create a Java program that provides information on various aviation topics using a switch statement. The program should be interactive and guide the user through different choices to retrieve specific details.

Core Requirements:

  1. User Menu: The program must first present the user with a numbered menu of aviation topics:
    • 1. Aircraft Type
    • 2. Flight Phase
    • 3. Airspace Class
    • 4. Weather Condition
  2. Outer switch Statement: Use a primary switch statement to handle the user's initial selection.
  3. Nested switch Statements: Inside each case of the main switch, implement a nested switch statement to handle a second layer of user input for more detailed information.
  4. Case-Insensitive Input: The program should handle user input for the nested switch statement without being case-sensitive (e.g., "boeing" or "BOEING").
  5. Error Handling: Both the outer and inner switch statements must include a default case to handle invalid user input and display an appropriate error message.
  6. Program Flow: The program should use a Scanner to read all user inputs and should be clear and easy to read.
